# Bomb Clicker Game (Scratch)

## Description

This is a simple time-based clicking game made in Scratch. The player has 30 seconds to click on a moving bomb sprite and score as many points as possible.

## How to Play

* Click the green flag to start the game.
* A bomb sprite will appear and move to random positions on the screen.
* Click the bomb to earn points.
* Each successful click adds 1 point to your score.
* The game ends after 30 seconds.

## Features

* 30-second countdown timer
* Random bomb movement
* Score tracking system
* Simple and fun gameplay

## Goal

Try to score as many points as you can before the time runs out!
